sheethub
========
SheetHub is a self-publishing platform to share and sell sheet music.
To make it easy for composers and arrangers to share their creations.

http://goo.gl/9DAcsw

## System requirements
'''
brew install <requirement>
'''

- elasticsearch
- libpng
- libjpeg
- imagemagick

```
brew install <requirement>
```

## Get Started

```
bundle install
rake db:migrate
rails s
```
